オレンジボタン — 3 種類だけ

指示はこの3つ＋白枠・グレーで足ります。実装は JPadOrangeChromeStyle.swift。

1. ORANGE-A — 待機・選択（形の正解 = DARK）

DARK NOTE HOLD
寸法高さ 44 · 角丸 14 · 文字 16pt heavy
色地 buttonIdleFill · 枠 buttonIdleBorder · 字 primaryLabel
使う場面PAD STYLE 選択中、NOTE 待機、HOLD オフ、SET オフ、EDIT オフ

2. ORANGE-G — 押下・オン（色の正解 = NOTE 押下、形は A と同じ）

NOTE Continue HOLD
寸法A と同じ（44 · 14 · 16pt heavy）
色地 accentGradient · 枠 白 28% · 字 buttonLabelFilled
使う場面NOTE 押し続け、Continue、HOLD オン、SET オン、EDIT 編集中

3. COMPACT — 小（OCT+ サイズ・両端丸）

OCT + OCT + TEST LOAD
寸法高さ 28 · カプセル（完全丸端）· 文字 12pt heavy
色オレンジは A / G と同じ。白枠のみ WHITE
使う場面上部 OCT±、設定 TEST NOTE / Buy / LOAD、メイン EDIT（小）

対照（オレンジ以外）

FLASH RESET

GREY 未選択 PAD STYLE · WHITE LOAD / RESET / 閉じる

メイン下部 HOLD/RESET は幅だけパッドセル幅・高さはレイアウト依存（52 等）でも 角 14・16pt heavy は A/G に合わせる。
